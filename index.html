Connect your Strava account

Strava's API needs an "application" to authorize against. Because this site has no server, you use your own free API application — your credentials stay in this browser.

  1. Open strava.com/settings/api and create an application (any name/category).
  2. Set Authorization Callback Domain to
  3. Copy the Client ID and Client Secret below.

Stored only in this browser's localStorage. This page never sends them anywhere except directly to strava.com.

⚠︎ Strava requires the app owner to have an active Strava subscription — without one the app shows as "inactive" and data requests fail. No subscription? Use the CSV export option instead; it works for everyone.
